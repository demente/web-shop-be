## Description

Simple web shop backend written with [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ yarn install
```

Generate database schema:

```bash
$ yarn migration:run:local
```


## Running the app

```bash
$ yarn start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
